# Glossary Notes

This file tracks glossary decisions for the webtoon adaptation.

---

---

## Current Campaign Rule

- Vesper Aurelius and Leo Gray are the main protagonists.
- Legacy characters, houses, factions, and locations remain valid background continuity.
- Legacy material should not overwhelm the opening webtoon arc.

---

## Update Protocol

When a term changes:

1. Update this file.
2. Update affected episode files.
3. Update the production bible if the term appears in global structure.
4. Add a note to the relevant file’s change log if the correction affects continuity.
